# 📅 Interactive Calendar Component

A polished, responsive wall-calendar inspired component built using React and Tailwind CSS.

---

## 🚀 Features

- 📸 Wall Calendar Aesthetic
  - Hero image with month/year overlay
  - Clean layout inspired by physical wall calendars

- 📆 Date Range Selection
  - Select start and end dates
  - Visual highlighting for selected range

- 📝 Notes Section
  - Write and persist notes using localStorage

- 🎯 Enhanced UX
  - Current day highlight
  - Weekend column styling (Sat/Sun)
  - Out-of-month days faded
  - Smooth hover animations

- 📱 Fully Responsive
  - Desktop → side-by-side layout
  - Mobile → stacked layout

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- date-fns
- clsx

---

## 📦 Installation

git clone <your-repo-link>
cd calendar-app
npm install
npm run dev
